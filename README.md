## Deploy to GitHub Pages (Web)

1) Push this repo to GitHub and ensure your default branch is `main`.
2) In GitHub: `Settings` → `Pages` → set **Build and deployment** to **GitHub Actions**.
3) Push to `main`. The workflow `.github/workflows/deploy_github_pages.yml` builds and deploys.
4) Your site will be available at:

`https://<your-username>.github.io/<your-repo-name>/`
